Калькулятор рецепта питательного раствора по профилю

Данный калькулятор позволит самостоятельно рассчитать рецепт питательного раствора для вашего растения. Для расчета вам необходимо иметь профиль питания растения. В качестве примера забит профиль для томатов.
Вы можете сохранять ваш расчет и возвращаться к нему позже, для этого есть соответстуюшие кнопки.
Для пересчета значений, необходимо совершить действие в любом поле ввода.

Как пользоваться?
  • В таблице профиля вам необходимо указать профиль питания вашего растения. Сделать это нужно в первой строке.
  • В первой строке таблицы соотношений у вас появится расчет соотношений элементов необходимый для конечного расчета количества нужных вам солей.
  • Во второй строке таблицы соотношений вам необходимо либо скопировать соотношения первой строки, либо произвести корректировку на ваше усмотрение.
  • Во второй строке таблицы профиля вы можете посмотреть профиль, который получится при смешивании настоящих солей.
  • Затем вам нужно выбрать необходимое количество раствора.
  • В таблице микроэлементов, вы можете установить свои значения для корректировки питания.
  • Выберите в каком соотношении будет замешиваться концентрат для приготовления раствора. И введите желаемый объем для емкостей содержащих раствор.
  • Далее вы увидите количество необходимых солей для приготовления раствора по вашему профилю и рекомендации.
     Какие элементы блокируют друг друга при избытке.
  • Калий блокирует: Кальций, Магний
  • Магний блокирует: Кальций
  • Кальций блокирует: Азот, Калий
     Помощь по соотношениям.
  • Данные рекомендации не являются обязательными и основаны на наблюдениях.
  • K:Mg < 10
  • 1 < K:Ca < 2
  • N:Ca < 1.5
  • K:Ca >= N:Ca

  • Томатный раствор
  • Сохранить рецепт
  • Загрузить рецепт
